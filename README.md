# Platformer
A simple platformer made in Godot.
## Quick start
### Users (windows only)
1. Download the .zip file
2. Unzip it (WARNING! Please keep the
.exe and .pck files in the same folder, otherwise the game breaks)
3. Run the .exe file
4. Enjoy
### Forkers and pull-requesters
1. Clone this repository to a suitable folder
2. Open Godot 4.4
3. On the top-left, press "Import"
4. Select the directory of the folder
5. The project is now editable
